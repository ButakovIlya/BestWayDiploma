# Архитектура хранения файлов

## Корневой каталог

Все файлы проекта - роутинг, компоненты, хуки и т.п. хранятся в `app`.

## Стратегия или архитектура хранения файлов

Файлы проекта разделяются по маршрутам или по общим признакам.

Эта стратегия хранит глобально общий код приложения в корневом каталоге `app` и разделяет более специфический код приложения на сегменты маршрута, которые их используют.
